# Documentary Streamer

A React.js application for streaming documentaries built with Express.js backend and modern React frontend.

## Features

- Modern React.js frontend with TypeScript
- Express.js backend API
- Real-time streaming capabilities
- Responsive design with Tailwind CSS
- Database integration with Drizzle ORM

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/afroz-ssi/documentary_streamer_reactjs.git
cd documentary_streamer_reactjs
```

2. Install dependencies
```bash
npm install
```

3. Start development server
```bash
npm run dev
```

The application will be available at `http://localhost:5000`

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run check` - Type checking
- `npm run db:push` - Push database schema

## Tech Stack

- **Frontend**: React.js, TypeScript, Tailwind CSS, Vite
- **Backend**: Express.js, Node.js
- **Database**: PostgreSQL with Drizzle ORM
- **Authentication**: Passport.js
- **UI Components**: Radix UI, Lucide React
